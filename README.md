# IZLA Lashes

Luxury lashes single-page storefront built with Next.js and a bilingual (FR/AR) UI.

## UI / UX

- **Layout flow:** announcement bar, language toggle, sticky header, hero with CTAs, features strip, product grid, how-to-order steps, testimonials, delivery highlight, and a dense footer.
- **Bilingual experience:** all key content switches between French and Arabic; Arabic uses RTL-aligned sections for readability.
- **Visual system:** warm luxury palette defined in `styles/izla.css` (noir, espresso, caramel, nude, ivoire). Typography uses Playfair Display (serif headlines), DM Sans (UI), and Noto Sans Arabic.
- **Interaction design:** primary and secondary CTAs, WhatsApp purchase links, product badges, and subtle hover feedback on buttons, cards, and footer links.
- **Responsive behavior:** mobile-first grid stacking, compact hero layout, hidden top nav on small screens, and single-column grids for steps/testimonials/delivery.
